# dbd-results

## アプリの概要 / Overview

Dead by Daylight の戦績を登録できるアプリです。キラー側、サバイバー側どちらも登録可能です。
戦績登録のタブから登録でき全体戦績、個人戦績、直近戦績を確認できます。
